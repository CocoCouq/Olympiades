## Project setup
### Install Node.js et npm
- Installer Node.js et npm:

    [Node.js](https://nodejs.org/en/)
    
    ###### Node.js automatically install npm
    
    [npm-info](https://www.npmjs.com/get-npm)
    
### Install npm
- Open project and install npm to launch project :

        npm install
        
### Build all assets
- To build assets use :

        npm run build:all
